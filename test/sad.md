## Installation

Para inslar el sdk de manera remota puede ejecutar:

```shell
npm install naranja_customer_api_testing --save
```

##### Local development

Para usar la libreria de manera local ejecute el siguiente comando: 

```shell
npm install file:path_libreria
```
